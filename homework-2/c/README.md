# Blöcke, Blöcke, Blöcke 

Verwende __yield__ für die folgenden Aufgaben: 
* definiere eine Funktion, die den übergebenen Block dreimal ausführt 
* definiere eine Funktion, die eine Zahl als Argument nimmt, und den Block dann so oft ausführt 
* erweitere die Funktion so, dass sie mit Time.now die Zeit stoppt, die die Ausführung gebraucht hat – benchmark funktion!
